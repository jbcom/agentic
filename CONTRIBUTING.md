# Contributing to agentic-dev-library

Thank you for your interest in contributing to our project!

## Code of Conduct

Please be respectful and inclusive in all your interactions with the community.

## How to Contribute

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Submit a pull request

## Development Setup

- Python: `pip install uv && uv pip install -e ".[dev]"`
- Node.js: `npm install`
- Rust: `cargo build`

## AI Assistance

We use AI agents to help with PR reviews and issue triage. You can trigger them by using commands in comments:
- `/jules <prompt>`: Delegate a task to Google Jules
- `/cursor <prompt>`: Delegate a task to Cursor Agent
- `@claude <prompt>`: Get interactive help from Claude

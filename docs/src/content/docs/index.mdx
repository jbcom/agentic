## What can you build with Agentic?

<CardGrid>
  <Card title="Spawn AI agent fleets" icon="rocket">
    Dispatch multiple AI agents across repositories. They clone, work, and open PRs — all in parallel, with automatic token routing per GitHub org.

    ```bash
    agentic fleet spawn "my-org/api" \
      "Fix the failing CI" --auto-pr
    ```
  </Card>

  <Card title="Orchestrate AI crews" icon="puzzle">
    Define crews in YAML, run them on CrewAI, LangGraph, or Strands. One format, any framework. Includes CLI runners for aider, claude-code, and ollama.

    ```bash
    agentic-crew run my-app analyzer \
      --input "Review auth module"
    ```
  </Card>

  <Card title="Triage issues at scale" icon="magnifier">
    Portable Vercel AI SDK tools for issue triage, code review, and sprint planning. Works with GitHub, Jira, Linear, and Beads.

    ```typescript
    const result = await generateText({
      model: anthropic('claude-sonnet-4-20250514'),
      tools: getTriageTools(),
      prompt: 'Triage all open critical bugs',
    });
    ```
  </Card>

  <Card title="Generate retro RPGs" icon="star">
    AI-powered game generation with Bevy. Procedural worlds, characters, quests, and assets — from a TOML config file.

    ```rust
    let game = QuickGenerator::new()
        .genre(Genre::Jrpg)
        .theme(Theme::Fantasy)
        .generate_and_run();
    ```
  </Card>
</CardGrid>

## Quick Start

<Tabs>
  <TabItem label="TypeScript (Fleet)">
    ```bash
    # Install the control package
    npm install -g @jbcom/agentic

    # Initialize configuration
    agentic init

    # Spawn your first AI agent
    agentic fleet spawn "my-org/my-repo" \
      "Fix the failing tests" --auto-pr

    # Monitor progress
    agentic fleet list --running
    ```
  </TabItem>
  <TabItem label="TypeScript (Triage)">
    ```typescript
    import { getTriageTools } from '@jbcom/agentic/tools';
    import { generateText } from 'ai';
    import { anthropic } from '@ai-sdk/anthropic';

    const result = await generateText({
      model: anthropic('claude-sonnet-4-20250514'),
      tools: getTriageTools(),
      maxSteps: 10,
      prompt: 'List critical bugs and create a triage plan',
    });

    console.log(result.text);
    ```
  </TabItem>
  <TabItem label="Python (Crews)">
    ```bash
    # Install with your preferred framework
    pip install agentic-crew[crewai]

    # Define a crew in YAML, then run it
    agentic-crew run my-package analyzer \
      --input "Review this codebase for security issues"
    ```

    ```python
    from agentic_crew import run_crew

    # Auto-detects best available framework
    result = run_crew("my-package", "analyzer",
                      inputs={"code": "..."})
    ```
  </TabItem>
  <TabItem label="Rust (Games)">
    ```bash
    # Build and run the game generator
    cargo run --bin vintage-game-generator

    # Or use it as a library
    ```

    ```rust
    use vintage_game_generator::prelude::*;

    fn main() {
        let game = QuickGenerator::new()
            .genre(Genre::Jrpg)
            .theme(Theme::Fantasy)
            .generate_and_run();
    }
    ```
  </TabItem>
</Tabs>

## Design Principles

<CardGrid>
  <Card title="Provider Agnostic" icon="setting">
    Every package works with multiple AI providers — Anthropic, OpenAI, Google, Mistral, Ollama, Azure. Swap providers without changing code.
  </Card>
  <Card title="Declarative Config" icon="document">
    Crews in YAML. Pipelines in JSON. Fleets from CLI flags. Configuration lives in version control, not in scripts.
  </Card>
  <Card title="Security First" icon="approve-check">
    Token sanitization, SHA-pinned Actions, OIDC publishing, sandboxed execution. No hardcoded credentials anywhere.
  </Card>
  <Card title="Polyglot by Design" icon="puzzle">
    TypeScript for orchestration. Python for AI frameworks. Rust for performance. Nx orchestrates the monorepo. Use what fits.
  </Card>
</CardGrid>
